# ML-DRIVEN-REAL-TIME-CLASSIFICATION-AND-PREDICTION-OF-DDOS-ATTACKS
Machine learning–based system that detects and classifies network attacks in real time using Random Forest (binary detection) and XGBoost (multi-class classification). It processes network traffic, assigns severity levels (Low/Medium/High), and displays alerts on a Flask dashboard. Achieved ~92% accuracy on the UNSW-NB15 dataset.
